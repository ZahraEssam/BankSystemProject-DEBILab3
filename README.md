# Bank System Project

A **JavaScript-based banking system simulation** that demonstrates core banking operations such as account creation, deposits, withdrawals, transfers, and security checks.

## Features
- Create and manage bank accounts
- Deposit and withdraw money
- Transfer between accounts
- Daily withdrawal limit
- Suspicious activity detection
- Password validation

## Technologies Used
- JavaScript
- HTML
- CSS
